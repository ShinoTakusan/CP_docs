3. C問題

ソースコード next_permutationでの実装

ソースコード DFSでの実装

順列、DFS：この問題は、基本的な考え方は一緒だが実装によって2通りある。

3.1. 考え方

この問題の共通している考え方について解説する。

この問題は、いかに効率よくすべての数列AAAを列挙するかが大切である。これからその方法を考える。まずはm = 4 , n = 3として考えてみよう。図のようにmをボール、nを棒として考えて見る。

次に4個のボールと、3個の棒を適当に並び変えてみる。

するとこんな感じになる。次に、棒より左にあるボールの数を数えてみる。

数列AAAの中の1つを表現していることが分かるだろうか？何が言いたいかというと、これらのボールと棒の並び方は、数列AAAの中の1パターンを表現している。これらのボールと棒の並び方は、数列AAAのどれかと1対1対応しており、このボールと棒の並び方を考えればよい。その並び方は、m+nCn{}_{m+n} C_n​m+n​​C​n​​となる。つまり、m+nの場所にn本の棒の置き方である。計算量はO(m+nCn)O({}_{m+n} C_n)O(​m+n​​C​n​​)で入力より、最低でも20C10{}_{20} C_{10}​20​​C​10​​(184756　※google検索やwolfram alphaなどで20 choose 10とすれば出てくる)なので間に合いそうである。※数列AAAを列挙するための計算量、実装の仕方で最悪計算量は少し変わる。(後述)

数列AAAをうまく列挙出来たら、あとは各条件に合うかどうかを調べるだけなので、数列の生成にO(N)O(N)O(N)、条件について調べるのがO(Q)O(Q)O(Q)、最終的にO((N+Q)m+nCn)O((N+Q) {}_{m+n} C _n)O((N+Q)​m+n​​C​n​​)となる。

3.2. next_permutaion での実装

数列AAAを作成するために、next_permutaionを利用して実装する。※next_permutaion とは配列の中身を元に辞書順の次の順列を生成する関数 next_permutaionについて ボールと棒はそれぞれ0と1で表現することが可能で、1が来るまでの0の個数を利用すれば上の考え方と同じそうに数列AAAを生成することができる。しかし、この実装では、AiA_iA​i​​が0のときも探索してしまうので、注意が必要。最悪計算量は20C10{}_{20} C _{10}​20​​C​10​​となる。（参考１）

3.3. DFS での実装

数列AAAを作成するために、DFS（再帰関数）を利用して実装する。始めに要素0の配列を準備し、その末尾に1を追加し、その末尾をインクリメントしていくことで数列AAAを生成していく。とりあえず、ソースコードを見たほうが分かると思う。それでもわからなければDFSするごとに配列の大きさを出力したり、実際に紙に書いてみると分かると思う。この実装ではAiA_iA​i​​が0のときを考慮しないので最悪計算量は19C9{}_{19} C _{9}​19​​C​9​​となる。始めに入れた1は、インクリメントされないので注意。（参考２）

4. D問題

ソースコード

数学：この問題は数学的考察を行うと簡単に解くことができる。まずは、小数部分と整数部分で分けてみる。今回は、次のように考察していくとうまくいく。

N := floor(xB)floor(\frac{x}{B})floor(​B​​x​​)の整数部分、M := floor(xB)floor(\frac{x}{B})floor(​B​​x​​)の小数部分とする。

xB=N+M\frac{x}{B} = N + M​B​​x​​=N+Mが成り立つ。

これより、AxB=AN+AMA\frac{x}{B} = AN + AMA​B​​x​​=AN+AM であり、floor(AxB)=floor(AN+AM)floor(A\frac{x}{B}) = floor(AN + AM)floor(A​B​​x​​)=floor(AN+AM)である。

AとNは整数なので floor(AN+AM)=AN+floor(AM)floor(AN + AM) = AN + floor(AM)floor(AN+AM)=AN+floor(AM)となる。

以上より、floor(AxB)−A×floor(xB)⟺AN+floor(AM)−AN⟺floor(AM)floor(\frac{Ax}{B}) - A \times floor(\frac{x}{B})\Longleftrightarrow AN + floor(AM) - AN \Longleftrightarrow floor(AM)floor(​B​​Ax​​)−A×floor(​B​​x​​)⟺AN+floor(AM)−AN⟺floor(AM)

以上の考察より、この問題はMの最大化を考えればよくその最大のMは x=B−1x = B-1x=B−1のときである。しかし問題の制約より、n以下であることが必要である。上の考察でも分かる通り、floor(AxB)−A×floor(xB)floor(\frac{Ax}{B}) - A \times floor(\frac{x}{B})floor(​B​​Ax​​)−A×floor(​B​​x​​) は広義単調増加関数なので、もしも、nがB-1以下であればnのときが最大となる。

以上をまとめると x=min(n,B−1)x = min(n,B-1)x=min(n,B−1)のときが答えである。(1)

このように、床関数(floor()または、⌊\lfloor⌊　⌋\rfloor⌋)やガウス記号([　])のときは、不等式を考えるか、その中身の整数部と小数部に分けて考察するとよい。
